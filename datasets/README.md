# Healthcare datasets

| File / folder | Use |
|---------------|-----|
| `heart_disease_uci.csv` | UCI heart disease — unified ML training + knowledge graph |
| `diabetes.csv` | Pima diabetes — knowledge graph relationships |
| `kidney_disease.csv` | CKD cohort — knowledge graph relationships |
| `symptom_disease.csv` | Symptom–disease co-occurrence edges |
| `cardio.csv` | Cardiovascular cohort (optional; `;` separator) — primary ML when present |
| `mimic-iii-clinical-database-demo-1.4/` | MIMIC-III demo ICU tables |
| `mimic_cohort.csv` | Generated by `python -m data.mimic_etl` from MIMIC demo |

## Regenerate artifacts

From repository root:

```bash
python -m backend.preprocessing_unified
python -m backend.training.train
python -m backend.knowledge_graph.pipeline
```
